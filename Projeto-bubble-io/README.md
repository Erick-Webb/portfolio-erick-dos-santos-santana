# 🤖 SM7 - Engenharia de Software com IA (Bubble.io)

## 📌 Descrição
Este projeto explora o desenvolvimento de software utilizando inteligência artificial e ferramentas no-code, com foco na plataforma Bubble.io para criação de aplicações de forma ágil, prática e eficiente.

## 🎯 Objetivo
Demonstrar como a integração entre inteligência artificial e plataformas no-code pode acelerar o desenvolvimento de sistemas, permitindo a criação de soluções funcionais sem a necessidade de programação tradicional.

## 🛠️ Tecnologias Utilizadas
- Bubble.io
- Inteligência Artificial
- Desenvolvimento No-Code / Low-Code
- Ferramentas de prototipação e automação

## 🧠 Conceitos Trabalhados
- Engenharia de Software com IA
- Desenvolvimento No-Code e Low-Code
- Automação de Processos
- Prototipação Rápida
- Estruturação de Fluxos e Lógica de Sistemas

## 📊 Conteúdo
- Estrutura da aplicação desenvolvida
- Funcionalidades implementadas
- Fluxos e lógica do sistema
- Integrações com recursos de IA *(quando aplicável)*
- Testes e validações realizadas

## 🔄 Metodologia
1. Definição do problema ou necessidade  
2. Planejamento da solução  
3. Desenvolvimento da aplicação no Bubble.io  
4. Implementação das funcionalidades  
5. Realização de testes e ajustes  

## 💡 Aprendizados
- Agilidade no desenvolvimento com ferramentas no-code
- Aplicação prática de inteligência artificial em sistemas
- Criação rápida de protótipos funcionais
- Importância da organização de fluxos e lógica de aplicação

## ▶️ Como Acessar
- Abra o projeto na plataforma Bubble.io *(se disponível)*  
- Navegue pelas funcionalidades implementadas  
- Teste os fluxos e interações da aplicação

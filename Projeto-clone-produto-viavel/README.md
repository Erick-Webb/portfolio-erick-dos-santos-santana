# 🚀 SM5 - Do Clone ao Produto Mínimo Viável (MVP)

## 📌 Descrição
Este projeto apresenta o desenvolvimento de um produto digital a partir da análise e replicação de uma solução existente (clone), evoluindo posteriormente para a criação de um Produto Mínimo Viável (MVP) com funcionalidades essenciais e foco na entrega de valor.

## 🎯 Objetivo
Compreender a estrutura e o funcionamento de um produto digital, reproduzindo suas principais funcionalidades e evoluindo a solução para um MVP funcional, simplificado e eficiente.

## 🧠 Conceitos Trabalhados
- Produto Mínimo Viável (MVP)
- Engenharia Reversa de Produtos
- Priorização de Funcionalidades
- Desenvolvimento Iterativo
- Estruturação de Sistemas

## 📊 Conteúdo
- Análise do produto original *(clone)*
- Identificação das funcionalidades principais
- Desenvolvimento do clone inicial
- Evolução para o MVP
- Estrutura do sistema desenvolvido
- Testes e ajustes realizados

## 🔄 Etapas do Projeto
1. Análise do produto existente  
2. Identificação das funcionalidades essenciais  
3. Desenvolvimento do clone inicial  
4. Refinamento e simplificação para MVP  
5. Realização de testes e melhorias  

## 💡 Aprendizados
- Foco no essencial para entrega de valor
- Importância da priorização de funcionalidades
- Aplicação prática de visão de produto
- Desenvolvimento progressivo e iterativo

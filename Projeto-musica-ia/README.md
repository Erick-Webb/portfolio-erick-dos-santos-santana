# 🎵 Projeto Música & IA

## 📌 Descrição do Projeto
Este projeto explora a interseção entre criatividade humana e Inteligência Artificial.

Utilizando técnicas de RAG (Retrieval-Augmented Generation) através do Gemini Gems, foi criado um assistente especializado na análise lírica de clássicos do Samba e Pagode.

O objetivo final foi produzir um EP com 5 faixas inéditas, cujas letras foram geradas por IA e transformadas em música utilizando o Suno AI.
